About Wordify

Wordify was inspired by this graph of rappers' vocabularies, and uses MusixMatch's lyrical data to score you on the variedness of lyrical content in your top songs.

How is my score calculated?

You're given a score of the average number unique words in the 15 wordiest songs in your top 30 Spotify songs of the last 6 months. So, the 15 songs in the top 30 which have the most varied lyrical content. Scores go up to 1000.

How did you make it?

The project is all open source; check out the github repo here. I used Spotify's API and MusixMatch's API (they license lyrics to Spotify).

Why does it only work for like 3 people a day?

The MusixMatch API's free tier is pretty limited for my needs; it offers 2000 API calls a day and each use makes arond 200 per scoring. That's still a decent amount of people, but if any requests get fudged or you reload the page or whatever, the number per user goes up. So, the service will be pretty limited until I feel like paying however much a month.
